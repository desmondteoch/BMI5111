### BMI5111 - Capstone Project
Thesis submission for the Master of Science in Biomedical Informatics

Components
1. Overseas_ML.ipynb - Code file for machine learning models to predict whether overseas regulatory action has been taken given case description
2. Overseas_BERT.ipynb - Code file for ClinicalBERT model to predict whether overseas regulatory action has been taken given case description
3. stat_RISMED.py - Code file to compile all models and generate predicted risk score for potential public health impact of health product defect
